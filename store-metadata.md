### Full description
- **Overview**  
  GitHub Widget is a minimal, always-available desktop companion for developers. It mirrors your real GitHub contributions calendar, streak, and activity stats in a compact, glassy widget that lives on your Windows desktop or stays pinned above your work.

- **Features**  
  - Live GitHub contribution heatmap for the last year.  
  - Total contributions, current streak, best day, and daily average.  
  - Optional personal access token support for more accurate data (with `read:user` scope).  
  - Always-on-top mode for true “widget” behavior.  
  - Tray icon with quick controls for size, opacity, and startup behavior.  
  - Remembers your position, size, and settings across restarts.  
  - All data stays on your machine; only GitHub’s official APIs are contacted.

- **Privacy**  
  GitHub Widget stores your settings locally on your device. No analytics, tracking, or external servers are used beyond GitHub’s own API.
