# Contributing to ContribTracker

Thank you for your interest in contributing! Here's how you can help.

## Getting Started

1. **Fork** the repository
2. **Clone** your fork:

   ```bash
   git clone https://github.com/YOUR_USERNAME/github-widget.git
   cd github-widget
   ```

3. **Install dependencies:**

   ```bash
   npm install
   ```

4. **Run in development mode:**

   ```bash
   npm run dev
   ```

## How to Contribute

### Bug Reports

- Use [GitHub Issues](https://github.com/akshithnallaginnela/github-widget/issues) to report bugs
- Include your OS version, steps to reproduce, and screenshots if possible

### Feature Requests

- Open an issue with the `enhancement` label
- Describe the feature and why it would be useful

### Code Contributions

1. Create a new branch: `git checkout -b feature/your-feature`
2. Make your changes
3. Test thoroughly
4. Commit with clear messages: `git commit -m "feat: add dark mode toggle"`
5. Push and open a Pull Request

## Code Style

- Use consistent indentation (4 spaces)
- Follow existing naming conventions
- Comment complex logic
